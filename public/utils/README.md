# Utilities Directory

يحتوي هذا المجلد على الدوال المساعدة (Helper Functions) والأدوات العامة.

## المحتوى:
- `translations.js`: نظام ترجمة الموقع.
- `image-loader.js`: أدوات تحميل ومعالجة الصور.
- `data-sync-fix.js`: إصلاحات وتزامن البيانات.

## القواعد:
- الدوال هنا يجب أن تكون "Pure Functions" قدر الإمكان.
- لا تحتوي على منطق خاص بصفحة معينة بل أدوات عامة للمشروع.
