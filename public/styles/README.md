# Styles Directory

يحتوي هذا المجلد على كافة ملفات التنسيق (CSS) الخاصة بالمشروع.

## الهيكل:
- `base.css`: التعريفات الأساسية والتصفير (Reset).
- `tokens.css`: المتغيرات والألوان (Design Tokens).
- `layout.css`: هيكل الموقع والعناصر المتكررة.
- `header.css`, `hero.css`: تنسيقات لمقاطع محددة.

## ملاحظات:
- استخدم المتغيرات المعرفة في `tokens.css` دائماً.
- اتبع منهجية التسمية الموحدة (مثل BEM) إذا أمكن.
