# Assets Directory

هذا المجلد يحتوي على كافة الملفات الثابتة (Static Assets) المستخدمة في المشروع.

## الهيكل الفني:
- `images/`: الصور العامة، صور المشاريع، وخلفيات الموقع.
- `icons/`: الأيقونات بصيغة SVG أو PNG.
- `fonts/`: الخطوط المخصصة المستخدمة في التصميم.

## ملاحظات:
- يرجى ضغط الصور قبل إضافتها لضمان سرعة تحميل الموقع.
- استخدم أسماء معبرة للملفات بصيغة `kebab-case`.
