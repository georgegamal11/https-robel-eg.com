# Pages Directory

يحتوي هذا المجلد على الأكواد التنفيذية الخاصة بالصفحات المختلفة.

## الصفحات الرئيسية:
- `script.js`: الصفحة الرئيسية (Home).
- `project-detail.js`: صفحة تفاصيل المشروع.
- `units.js`: صفحة الوحدات.
- `admin-*.js`: لوحة التحكم.

## هيكلية العمل:
- يتم ربط هذه الملفات بملفات HTML الموجودة في مجلد `public/`.
